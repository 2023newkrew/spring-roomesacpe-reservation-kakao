## 기능 목록

### 예약 관리
- [x] 예약 생성
  - [x] 타임테이블에 알맞는 시간대에 대해서만 예약 생성을 허락할 것
    - [x] 타임테이블에 없는 시간대라면 예외처리 할 것 (ErrorCode: 400)
  - [x] 같은 시간에 예약있으면 예외처리 할 것 (ErrorCode: 409)
- [x] 예약 조회
  - [x] id에 대응되는 예약이 없다면 예외처리 할 것 (ErrorCode: 404)
- [x] 예약 취소

### 테마 관리
- [ ] 테마 생성
- [ ] 테마 전체 조회
- [ ] 테마 단일 조회
- [ ] 테마 수정
   - [ ] 예약과 관련이 있는 테마일 시, 수정 불가능
- [ ] 테마 삭제
  - [ ] 예약과 관련이 있는 테마일 시, 삭제 불가능
